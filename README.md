```python
def predict(test_df, model, features, base_col):
    X = test_df[features]
    pred_res = model.predict(X)           # 预测残差
    pred = pred_res + test_df[base_col]   # 最终预测 = 残差 + 基线
    return pred
```

调用模型进行预测

#### 第四步：评估

```python
def evaluate(y_true, y_pred):
    mae = mean_absolute_error(y_true, y_pred)
    rmse = np.sqrt(mean_squared_error(y_true, y_pred))
    mape = np.mean(np.abs((y_true - y_pred)/(y_true+1e-5)))
    return mae, rmse, mape
```

运行后得到的mae, rmse分别在2-3和3-4之间，是可接受的（需要注意分辨这里的mae, rmse是根据哪些数据得出的，是否为全局数据）

这里的mape由于实际数据（作为分母）中有多个值为0，导致数值过于巨大，参考价值不高，后续删除
